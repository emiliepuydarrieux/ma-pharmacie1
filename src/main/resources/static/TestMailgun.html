🧪 Test Mailgun - Alerte Stock Fournisseurs

📋 Fonctionnalité

Cet outil teste l'envoi automatique d'emails aux fournisseurs quand le stock tombe en dessous du niveau de réapprovisionnement.

Condition : Si UNITES_EN_STOCK < NIVEAU_DE_REAPPRO → Email envoyé à chaque fournisseur de cette catégorie

ℹ️ Configuration Mailgun :
Assurez-vous que les variables d'environnement sont configurées :
  • MAILGUN_API_KEY
  • MAILGUN_DOMAIN
  • MAILGUN_FROM_EMAIL

🚀 Déclencher les Alertes

Cliquez sur le bouton pour vérifier tous les médicaments et envoyer les alertes :

📊 Informations de Test

⚠️ Données de test actuelles :
• Tous les médicaments en stock sont > niveau de réapprovisionnement
• Pour voir des alertes, modifiez les données de la BD (baissez UNITES_EN_STOCK ou augmentez NIVEAU_DE_REAPPRO)

🔧 Étapes pour Tester Localement

  1. Modifiez les données : 
    UPDATE MEDICAMENT SET UNITES_EN_STOCK = 5 WHERE REFERENCE = 1;
    Cela baisse le stock du premier médicament pour déclencher une alerte.
  2. Redémarrez l'application ou rechargez la page
  3. Cliquez sur le bouton ci-dessus pour déclencher l'envoi
  4. Vérifiez les logs de l'application pour les messages d'envoi
  5. Contrôlez Mailgun :
    • Allez sur app.mailgun.com
    • Sélectionnez votre domaine
    • Onglet "Logs" pour voir l'historique des emails envoyés
